# JSG Sangini Members Directory

A digital member directory system developed for JSG Sangini Main Bhilwara.

## Features

- Member profile cards
- Photo directory
- WhatsApp & Call integration
- Search members
- Blood group search
- Birthday & anniversary filters
- Alphabetical sorting
- Local cache for instant loading
- Google Sheet based backend

## Technology Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Google Apps Script
- Google Sheets
- Google Drive

## Architecture

Google Sheet
↓
Google Apps Script API
↓
Frontend Application
↓
Local Storage Cache
